Campus Navigation & Resource Management System is a C-based project that demonstrates the real-world application of Data Structures
to solve common campus problems. The system models the campus as a graph where buildings are nodes and paths are edges, enabling shortest-path navigation using BFS.
It also provides a structured resource booking mechanism using queues for normal requests and priority queues for urgent bookings,
ensuring fair and efficient allocation. Student records are managed using a Binary Search Tree for fast searching, while hash tables allow 
instant lookup of resource locations. Sorting algorithms help organize resources efficiently, and file handling ensures data persistence.
Overall, this project showcases how core data structures like graphs, trees, queues, hashing, and files can be integrated into a practical 
and scalable campus management system.
